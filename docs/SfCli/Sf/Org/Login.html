#login_access_token(instance_url:, target_org: nil) ⇒ Object

Note:

This method doesn’t support user interactive mode, so SF_ACCESS_TOKEN environment variable must be set before call this method.

Login to the org by access token.

How to set env variable outside of Ruby
In Unix/mac

$ SF_ACCESS_TOKEN=‘xxxxxxxxxx’ $ ruby app_using_sfcli.rb

OR

$ SF_ACCESS_TOKEN='xxxxxxxxxx' ruby app_using_sfcli.rb
In Windows

$ set SF_ACCESS_TOKEN=xxxxxxxxxx $ ruby app_using_sfcli.rb

Examples:

ENV['SF_ACCESS_TOKEN'] = 'xxxxxxxxxx'
sf.org.login_access_token instance_url: https://hoge.bar.baz.salesforce.com

Parameters:

  • target_org (Symbol, String) (defaults to: nil) —

    an alias of paticular org, or username can be used

  • instance_url (String) —

    URL of the instance that the org lives on.
